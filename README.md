## Setup

### One-Line Setup

```bash
git clone https://github.com/your-username/AetherionAI.git
cd AetherionAI
bash setup.sh
```

---

## Commands

### Backend – Veil of the Grove (CLI)

```bash
cd aetherion
python veil_of_the_grove.py cloak     # Cloak the veil
python veil_of_the_grove.py unveil    # Unveil the mysteries
```

### Root Scripts

```bash
yarn install
yarn start:expo
yarn start:backend
```

### Mobile App

```bash
cd frontend
yarn start
yarn lint
yarn format
yarn build
```
